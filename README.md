# HTB Initilisation script

## Usage
` ./recon.sh <IP> <DOMAIN> <DIRECTORY_NAME>`

- IP: HTB ip for machine
- DOMAIN: The domain name if applicable
- DIRECTORY_NAME: User choice to set up directory

## Requirements
- Tmux
- Nvim
